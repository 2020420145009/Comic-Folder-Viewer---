## 项目背景
你是否厌倦了在文件管理器和图片查看器之间来回切换？这个Python工具专为漫画/图集阅读优化，提供沉浸式全屏体验，智能识别文件夹结构，助你快速定位想看的章节！

## 使用要求
1.该工具用用于浏览漫画。

2.你的目录应该只有两类：A只有文件夹的目录；B只有图片的目录。

3.请将该文件放在漫画或图集的根目录下使用。根目录应为A类目录。

## 核心功能
直接展示图片文件夹 (自动检测jpg/png/webp)
显示子文件夹目录 (支持嵌套层级)

## 沉浸式体验
全屏显示模式

标题栏显示当前章节名称

## 快捷操作
↑ ↓ 键: 滚动翻看
